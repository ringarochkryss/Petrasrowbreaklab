Row Break in Chrome

  • Avoid floting elements- Several Boostrap classes will breaking this function (col-12 for instance)
  • All elements up in the DOM must have position:relative
  • Use break-before instead of page-break-before (or both)
  • The break-before must be on a block element
  • The break-before element must have a specified height
