Row Break in Chrome

  • Avoid floting elements,flex items and position:absolute.
  • Avoid Boostrap several BS classes will break this function (col-12 for instance)
  • All elements in DOM must have position:relative
  • Use break-before instead of page-break-before (or both)
  • The break-before must be on a block element
  • The break-before element must have a specified height
  • The break-before element must have a content, it can't be empty.
